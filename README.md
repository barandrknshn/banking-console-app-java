# Java Banking Console Application

Bu proje, Java öğrenme sürecimde temel programlama ve nesne yönelimli programlama kavramlarını pekiştirmek amacıyla geliştirdiğim konsol tabanlı basit bir bankacılık simülasyonudur.

Proje; müşteri işlemleri, hesap oluşturma, para yatırma/çekme, ödeme isteme, faiz getirisi hesaplama, döviz hesaplama ve bildirim simülasyonu gibi temel bankacılık senaryolarını içermektedir.

## Projenin Amacı

Bu uygulama, gerçek bir bankacılık sistemi geliştirmekten ziyade Java programlama dilindeki temel yapıların pratik edilmesi amacıyla hazırlanmıştır.

Bu proje sayesinde aşağıdaki konular üzerinde çalışma yapılmıştır:

- Class ve object yapısı
- Method kullanımı
- Getter ve Setter yapısı
- Scanner ile veri girişi alma
- Switch-case ile menü yönetimi
- ArrayList ve List kullanımı
- Basit doğrulama kontrolleri
- Kalıtım ve abstract class kullanımı
- Konsol tabanlı uygulama geliştirme

## Özellikler

- Müşteri ekleme
- Müşteri listeleme
- Hesap oluşturma
- Hesap listeleme
- Para yatırma
- Para çekme
- Ödeme isteme
- Faiz getirisi hesaplama

## Uygulama Akışı

Uygulama çalıştırıldığında konsol üzerinden menü gösterilir. Menü aracılığıyla müşteri ekleme, hesap oluşturma, para yatırma, para çekme, ödeme isteme, faiz getirisi hesaplama ve döviz hesaplama gibi işlemler yapılabilir.

Kullanıcı, yapmak istediği işlemin numarasını girerek ilgili bankacılık işlemini konsol üzerinden simüle eder.

## Örnek Menü
